# Effective Home and Bodyweight Training

Training without a gym is fully viable for building strength and fitness.
Progressive overload is still possible with bodyweight.

## Progressive Overload Without Weights

### Methods
1. Increase reps (push-ups: 3x10 → 3x15 → 3x20)
2. Decrease rest time (same work in less time)
3. Increase range of motion (full ROM push-ups → archer push-ups)
4. Progress to harder variations (push-up → pike push-up → handstand push-up)
5. Add tempo (3-second eccentric, 1-second pause, explosive concentric)
6. Single-limb progressions (squat → pistol squat, push-up → one-arm push-up)

## Full Body Home Workout Program

### Equipment: None required

#### Push Exercises (Chest/Shoulders/Triceps)
- Push-ups (standard → wide → diamond → archer → pseudo-planche)
- Pike push-ups → Handstand push-ups (wall-assisted)
- Dips between chairs

#### Pull Exercises (Back/Biceps)
- Doorframe rows (improvised)
- Towel rows
- Table rows
- Pull-ups if bar available

#### Lower Body
- Bodyweight squats → Jump squats → Pistol squat progression
- Reverse lunges → Bulgarian split squats
- Glute bridges → Single-leg glute bridges → Hip thrusts

#### Core
- Plank → Plank variations → Ab wheel (if available)
- Hollow body hold
- Dead bug
- Mountain climbers

### 3-Day Full Body Program (Home)
Day 1 and Day 3:
- 4x max push-ups
- 4x15 glute bridges
- 3x10 each leg reverse lunge
- 3x30s plank

Day 2:
- Active recovery: 30 min walk or yoga

## Resistance Bands (Optional but High Value)
- Bands add variable resistance — harder at end range
- Can replicate cable exercises
- Resistance band pull-aparts for shoulder health
- Banded squats for glute activation
- Cost: £10-25 for a set

## Cardio at Home
- HIIT: 20s work / 10s rest, 8 rounds of: burpees, jump squats, mountain climbers
- Jump rope: best calorie-per-minute home cardio
- Stair climbing: excellent LISS cardio if available
- Walking: underrated, easy to accumulate 8000+ steps daily
